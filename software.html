Software

Below, you'll find an overview of the software tools I have experience with, which I utilize for my projects.

Software Category Notes
Solidworks Design I use a Solidworks hobby license for designing parts and assemblies.
3Dexperience Design Used in conjunction with solidworks, 3DX is used to store design data, manage revisions and part lifecycles.
Autodesk Inventor Design Used for designing parts and assemblies.
Cura Manufacturing Generates GCode from an STL files for 3D printing.
Notepad++ Text Editor I use notepad++ to create & maintain this website.
DesignSpark PCB Design Electrical schematic and PCB design tool
LaserGRBL Manufacturing Used for sending GCODE to my laser engraver.
FlatCAM Manufacturing Used to generate toolpaths for PCB milling and other flat materials.
Arduino IDE Programming Used for any Arduino based projects and for configuring the firmware for all of my 3D Printers.
Fritzing Design Used for breadboard layouts. I used this for PCB design before swicthing to DesignSPark PCB.
