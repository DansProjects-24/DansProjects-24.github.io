  • VW Engine Table

This is a simple coffee table I made using the old crank case from a 1967 VW Beetle. The design is simple and requires very few parts and no use of a welder. All hardware uses existing holes in the engine block so no modifications are required should you wish to return the crank case to service inb a vehicle. In this case, the engine had lost oil pressure resulting in overheating and ceasing up. The crank journals had worn beyond salvaging but not wanting to scrap this iconic engine block it was given a new leas3e of life. If youd like to build your own VW engine block table, all the infoirmation needed can be found in the downloads [bookmark here] section.
